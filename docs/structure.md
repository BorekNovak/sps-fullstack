# Struktura projektu

```
sps-fullstack/
│   README.md
│   .gitignore
│   package.json
│   server.js
│
├───public
│       index.html
│       rezervace.html
│       assets/
│
├───api
│       api.js
│
└───database
        db.js
        schema.sql
        seed.js
        seed.sql
```

- `public/` – statické soubory (frontend)
- `api/` – backendové REST API
- `database/` – databázové skripty a připojení
- `server.js` – hlavní spouštěcí soubor
